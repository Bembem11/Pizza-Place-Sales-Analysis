## Datasets Used for Analysis
This project analysis uses four CSV files containing one year of sales data.
- Orders: Contains the `order_id`, date, and time for every transaction.
- Order Details: Tracks the specific `pizza_id` and quantity for each order.
- Pizzas: Provides the price and size for each pizza variant.
- Pizza Types: Includes the name, category, and ingredients for each pizza.
